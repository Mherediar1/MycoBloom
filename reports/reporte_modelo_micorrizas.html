Conclusión final

El modelo Random Forest presentó el mejor desempeño general durante las pruebas realizadas, alcanzando un Accuracy de 97.28%, Precision de 96.81%, Recall de 97.39%, F1-score de 96.80% y un error de 2.72%.

De acuerdo con el análisis de entrenamiento y prueba, el diagnóstico del modelo fue: Buen ajuste. Esto indica que el modelo seleccionado presenta un comportamiento estable y una adecuada capacidad de generalización sobre datos no observados.

Nota final

Para el entrenamiento del modelo se utilizaron únicamente las variables que estarán disponibles en la interfaz web, permitiendo que el modelo aprenda utilizando información real ingresada por el usuario.

Para el desarrollo de los modelos de aprendizaje automático, el conjunto de datos fue dividido en: 80% para entrenamiento (1024 registros) y 20% para pruebas (257 registros), manteniendo la distribución de clases mediante una partición estratificada.

Posteriormente, sobre el conjunto de entrenamiento se aplicó validación cruzada estratificada de 5 particiones (5-Fold Cross Validation), utilizando aproximadamente el 20% de los datos de entrenamiento para validación en cada iteración.

La evaluación se realizó mediante validación cruzada estratificada y métricas de desempeño como Accuracy, Precision, Recall, F1-score y Error.
